Peek Pop

集成预览、搜索和稍后阅读于一体的轻量浏览器扩展。

获取 Firefox 版 获取 Chrome 版 获取 Edge 版 GitHub 发布

安装

快速 设置指南

1

安装扩展

选择您的浏览器，并使用上面的下载按钮从官方商店安装 Peek Pop。

2

访问选项页面

安装后，建议访问选项页面以自定义设置并根据需要调整插件。

3

配置设置

根据您的偏好设置黑名单、自定义快捷键并调整预览行为。

主要特性

页面预览

即时预览网页，而无需离开当前选项卡。在决定访问之前快速查看内容。

搜索功能

快速搜索并找到您所选内容中确切需要的内容。

稍后阅读

将页面保存以供稍后阅读，永远不会失去对重要内容的跟踪。

灵活触发

通过双击、拖动、悬停、点击激活预览，并可以选择使用修饰键。

可定制设置

通过选项页面提供广泛的自定义选项，以根据您的特定工作流程量身定制扩展。

黑名单支持

支持多种格式，包括正则表达式、通配符和纯文本，以排除特定网站的预览。

使用指南

黑名单格式示例

使用以下格式配置要排除在预览之外的网站：

正则表达式格式:
/^https:\/\/example\.com\/.*$/

通配符格式:
https://example.com/*

纯文本格式:
https://example.com

常见模式示例
文件:
/\.(zip|rar|7z|exe|msi|md|pdf|docx?|xlsx?|pptx?|apk|dmg|iso)(\?.*)?$/

图片:
/\.(jpg|jpeg|avif|png|svg|ico|webp|gif)(\?.*)?$/

视频:
/\.(mp4|mkv|rmvb|rm|avi|ts|mov|flv)(\?.*)?$/

音频:
/\.(mp3|flac|ogg|wav|aac)(\?.*)?$/

下载链接:
/^(magnet:\?xt=urn:[a-z0-9]+:[a-f0-9]{32,40}.*|https?:\/\/[^\s]+\/dl[a-zA-Z0-9\/%+=_-]+)$/

已知问题

在 Firefox 上

当系统缩放未设置为 100% 时，弹出窗口在出现时可能会短暂闪烁。

在 macOS 上

预览弹出窗口在全屏模式下无法正常工作。为了获得最佳效果，请在非全屏模式下使用该窗口。
